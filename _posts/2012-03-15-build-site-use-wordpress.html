--- layout: post title: WordPress建站 tags: - web - WordPress status: publish type: post published: true meta: _edit_last: '1' --- 近来又拾起 WordPress 来建站玩玩。 介绍几个不错的插件：WordPress： 使用的最新的3.3.1
  • Theme： roots这是一个应用了Bootstrap的强大的CSS框架，由Twitter开源出来，采用它就可以做出漂亮的站点，详细请到github查阅文档
  • Plugins：
    • WP-SyntaxHighlighter 强大的代码高亮插件，可选多种高亮着色方案，包括eclipse着色方案(最爱)；
    • WP-PageNavi分页插件，结合Wp Pagenavi Style可以定制出不错的效果；
    • Flickr Manager 强大的插件，可以将Flickr作为图床，直接从WordPress上传到Flickr，并可插入到文章中
    • WP的插件社区还是相当活跃的，可以自己进行研究。
